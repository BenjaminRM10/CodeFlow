# Programming Training App

Develop muscle memory and typing speed for programming. Practice code and terminal commands with AI-generated lessons and visual keyboard guidance.

## Project info

This project is designed to help you master programming through daily typing practice with AI-powered code lessons.

## How to run locally

**Prerequisites**: Node.js & npm installed.

1.  Clone the repository.
2.  Navigate to the project directory.
3.  Install dependencies:
    ```sh
    npm install
    ```
4.  Start the development server:
    ```sh
    npm run dev
    ```
